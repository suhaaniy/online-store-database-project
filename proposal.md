Online Store Database Project Proposal

- This project is an Online Store Database System. The system will manage customers, products, orders, and payments.

Database will allow:
-store customer information
-managing product inventory
-creating orders
-tracking payments 

Main Entities 
-Customer
-Product
-OrderItem
-Payment

-A customer can place many orders
-An order can contain mulitple products
-A product can appear in many orders
-Each order has one payment
